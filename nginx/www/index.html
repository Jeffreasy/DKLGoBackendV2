De Koninklijke Loop API

Welkom bij de API van De Koninklijke Loop. Deze pagina biedt een overzicht van de beschikbare endpoints.

Publieke Endpoints

POST /api/contact

Verwerk een contactformulier inzending.

POST /api/aanmelding

Verwerk een vrijwilliger aanmelding.

Authenticatie Endpoints

POST /api/auth/login

Authenticeer een gebruiker en ontvang toegangstokens.

POST /api/auth/refresh-token

Vernieuw een verlopen toegangstoken met een refresh token.

POST /api/auth/forgot-password

Start het wachtwoord reset proces.

POST /api/auth/reset-password

Reset een wachtwoord met een reset token.

Beveiligde Endpoints (Admin)

Deze endpoints vereisen authenticatie en admin rechten.

GET /api/emails

Haal alle emails op.

GET /api/emails/stats

Haal email statistieken op.

PUT /api/emails/:id/read

Markeer een email als gelezen.

GET /api/contacts

Haal alle contactformulieren op.

PUT /api/contacts/:id/status

Werk de status van een contactformulier bij.

GET /api/aanmeldingen

Haal alle aanmeldingen op.

GET /api/aanmeldingen/stats

Haal aanmelding statistieken op.

GET /api/aanmeldingen/:id

Haal een specifieke aanmelding op.
